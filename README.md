### 🗂️ Second Extended Filesystem (ext2) — Full Stack
Full implementation of the **ext2 filesystem** with a client-server architecture:
- **Frontend** in React (deployed on **AWS S3**)
- **Backend** in Go (deployed on **AWS EC2**)
- REST API connecting both layers, simulating real filesystem operations

`Go` `React` `AWS EC2` `AWS S3` `REST API` `Systems Programming`
